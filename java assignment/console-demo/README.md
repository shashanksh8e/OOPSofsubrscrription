# Console Demo Module

## Files in this module:
- `UserHierarchyDemo.java` - Main console demonstration
- `ConsoleInterface.java` - Console utility functions
- `compile-console.bat` - Compile console components
- `run-console.bat` - Run console demo

## Dependencies:
- Core classes (User, RegularUser, AdminUser)
- Core interfaces (Manageable, Notifiable)
- Core enums (SubscriptionStatus, PaymentResult)

## How to run:
```bash
compile-console.bat
run-console.bat
```

## OOP Concepts Demonstrated:
- Constructor chaining
- Polymorphism
- Method overriding
- Interface implementation
- Dynamic method dispatch
