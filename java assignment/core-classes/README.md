# Core Classes Module

## Files in this module:
- `User.java` - Abstract base class
- `RegularUser.java` - Basic user implementation
- `AdminUser.java` - Admin user implementation
- `Manageable.java` - Management interface
- `Notifiable.java` - Notification interface
- `Billable.java` - Billing interface
- `Interactive.java` - UI interaction interface
- `SubscriptionStatus.java` - Status enumeration
- `PaymentResult.java` - Payment result enumeration

## Purpose:
This module contains the core business logic and OOP implementations that are shared by both console and web modules.

## OOP Concepts Implemented:
- Abstract classes and methods
- Single inheritance
- Interface implementation
- Method overriding
- Constructor chaining
- Encapsulation
- Polymorphism
- Enumerations

## Usage:
These classes are used as dependencies by:
- Console Demo Module
- Web Interface Module
